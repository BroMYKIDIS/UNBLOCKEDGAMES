# Unblocked.io 🎮

A simple, all-in-one gaming hub with favorites, trending games, and admin control.

## Features

- **Home Page** - View trending games
- **Games Library** - Browse all games and mark favorites
- **Favorites** - Quick access to your favorite games
- **Blog** - Information about the site
- **Contact** - Get in touch
- **Admin Panel** - Add new games to the hub

## How to Use

1. Open `index.html` in your browser
2. Navigate through pages using the top menu
3. Go to **Admin** to add new games (enter game name and URL)
4. Click **Play** to open a game or **⭐** to favorite it
5. View **Trending** games based on play count

## Data Storage

All data is stored locally in your browser using `localStorage`:
- Games list
- Favorites
- Play statistics

## Getting Started

Simply open the `index.html` file in your web browser. No server or installation required!
